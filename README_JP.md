# ESP32 おにぎり握り機 (Onigiri Maker) 🍙

このプロジェクトは、スマホからESP32に接続してサーボモーターを制御する「おにぎり握り機」のプログラムです。

## 機能
1. **WiFi接続**: `ESP32-Servo` (パスワードなし)
2. **自動ログイン**: WiFi接続後、自動的に操作画面が開きます（Captive Portal）。
3. **おにぎりモード**: 「握る！」ボタンを押すと、サーボが `0°` → `270°` → `0°` と動き、握る動作を再現します。
4. **手動モード**: スライダーを使って、好きな角度に調整できます。
5. **ゆっくり回転モード**: 指定した角度まで指定した秒数かけてゆっくり回転させることができます。

## 配線 (Wiring)
| ESP32 Pin | サーボモーター |
| :--- | :--- |
| **GPIO 26** | 信号線 (Orange/Yellow) |
| **5V or 3.3V** | 電源 (Red) |
| **GND** | GND (Brown/Black) |

※ サーボの電源は、使用するサーボの仕様に合わせてください（SG90などは5V推奨ですが3.3Vでも動くことがあります）。

## 書き込みトラブルシューティング
書き込み時に `Connecting........` と出たまま進まない、または `Wrong boot mode` エラーが出る場合：

1. 書き込みが始まったら（`Connecting...`が出たら）、ESP32本体にある **BOOTボタン** を押し続けてください。
2. 書き込みが開始されたら（ゲージが進み始めたら）、ボタンを離してOKです。
